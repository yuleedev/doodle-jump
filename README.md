# doodle-jump
the ever so classic doodle jump game... but with only one key?!
made for https://onekey.hackclub.com/ using unity and c#. all art drawn by me!

[will insert itch.io link here when finished]

## how to play
like the original, you:
- jump automatically if you touch a platform
- come in from the left side if you go out of the right side and vice versa
- die if you go below the camera

unlike the original, you:
- slowly drift left and hold the spacebar to move right!

it's a bit uncomfortable at first, but that's how i ended up
turning this game into one-key-only
